You can install the Astroroute CLI globally using npm. This is the recommended way to install Astroroute as it allows you to run the `astroroute` command from anywhere in your terminal.

## Prerequisites

- [Node.js](https://nodejs.org/) (version 18 or higher)
- npm (comes with Node.js)

## Install via npm

Run the following command in your terminal:

```bash
npm install -g astroroute
```

## Verify Installation

To verify that Astroroute has been installed successfully, run:

```bash
astroroute --version
```

You should see the version number of the installed CLI.

## Updating

To update Astroroute to the latest version, simply run the install command again:

```bash
npm install -g astroroute@latest
```

## Uninstalling

If you need to remove Astroroute from your system:

```bash
npm uninstall -g astroroute
```
